## Preject Overview

* Built a full-stack platform for developers to connect, network, and collaborate on projects, enabling seamless project discovery and teamwork.

* Implemented secure authentication using JWT and OAuth, ensuring user data protection and safe login flows across the platform.

* Integrated Razorpay payments and configured webhooks to handle real-time payment updates, supporting smooth financial transactions.

* Developed a real-time chat system with WebSockets, allowing users to communicate instantly and collaborate efficiently within projects.

* Automated recurring tasks using cron jobs, improving operational efficiency and reliability for backend processes.

* Deployed the application on AWS EC2, configured a custom domain, and utilized AWS SES for sending automated transactional emails.

* Designed the backend with a microservices architecture, allowing better scalability, maintainability, and modular service management.

* Managed application data using MongoDB Atlas, ensuring secure, cloud-hosted, and scalable database storage.
